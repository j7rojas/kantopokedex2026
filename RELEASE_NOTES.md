# Release Notes

All notable changes to this project will be documented in this file.

## v0.3.0 - 2026-03-16
- Feature: Added Evolutionary Chain section to Pokémon cards.
- Data: Implemented backend evolution data fetching using species and evolution chain endpoints.
- UI: Created EvolutionChain component with Generation III sprites and readable trigger labels.
- UI: Refactored Pokémon card layout so only the moves table scrolls internally.
- UI: Improved spacing and layout consistency across Base Stats, Evolutionary Chain, and Moves sections.
- UI: Rebuilt the grid/card layout using cleaner CSS Grid and Flexbox architecture.
- Refactor: Removed previous layout hacks and simplified the component structure.
- Documentation: Updated README to reflect the improved architecture.

## v0.1.0 - 2026-01-01
- Initial release with basic Pokémon info, base stats radar chart, and move list.
